# 阶段旅程海报单次生成风格补充

- 形式：完整成品海报，不是背景底图，不是信息图。
- 骨架：单一路径推进，整体从下往上或从左下到右上展开，像一路向北慢慢走。
- 顺序：城市顺序靠空间关系和场景过渡表达，不靠文字标签或编号。
- 内容：每个城市只保留 1 到 2 个最关键的意象，避免景点堆砌和拼贴感。
- 过渡：相邻城市之间必须通过水汽、风、路、石阶、地貌、色温变化自然连接，不能像三张图拼在一起。
- 版式：画面中心要有呼吸感，不能被桥梁、城墙、荷叶、大建筑横向堵死。
- 禁区：不要地图、不要中轴时间线、不要白底卡片、不要规则标签、不要明显中文标题、不要大段可读文字。
- 氛围：吉卜力绘本风格、旅行的青蛙审美、透明水彩晕染、低饱和莫兰迪色、细铅笔线稿、明显纸张纹理、2D 扁平极简。
